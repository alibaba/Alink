# 批式数据打印 (PrintBatchOp)
Java 类名：com.alibaba.alink.operator.batch.utils.PrintBatchOp

Python 类名：PrintBatchOp


## 功能介绍
该组件打印表中数据。

## 代码示例
### Python 代码
```python
from pyalink.alink import *

import pandas as pd

useLocalEnv(1)

df = pd.DataFrame([
       ["0,0,0"],
       ["0.1,0.1,0.1"],
       ["0.2,0.2,0.2"],
       ["9,9,9"],
       ["9.1,9.1,9.1"],
       ["9.2,9.2,9.2"]
])

# batch source 
inOp = BatchOperator.fromDataframe(df, schemaStr='y string')

inOp.print()

```
