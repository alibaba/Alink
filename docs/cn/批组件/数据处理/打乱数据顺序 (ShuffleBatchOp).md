## 功能介绍
该组件打乱数据的顺序。

## 代码示例
### Python 代码
```python
from pyalink.alink import *

import pandas as pd

useLocalEnv(1)

df = pd.DataFrame([
       ["0,0,0"],
       ["0.1,0.1,0.1"],
       ["0.2,0.2,0.2"],
       ["9,9,9"],
       ["9.1,9.1,9.1"],
       ["9.2,9.2,9.2"]
])
     
inOp = BatchOperator.fromDataframe(df, schemaStr='Y string')

inOp.link(ShuffleBatchOp()).print()

```
